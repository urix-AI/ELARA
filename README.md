# Project Elara 🎙️✨

Project Elara is a lightweight, responsive, voice-first AI assistant integrated directly into a web application's sidebar. Powered by the **DeepSeek API**, Elara allows users to interact with websites naturally through speech—eliminating the need for complex menu navigation or tedious typing. 

Designed for seamless integration into B2B SaaS platforms and e-commerce applications, Elara transforms standard web browsing into an intuitive, real-time conversational experience.

## 🚀 Features

*   **Voice-First UI:** Seamless Speech-to-Text (STT) and Text-to-Speech (TTS) pipeline for natural, low-friction conversations.
*   **Persistent Sidebar Design:** A clean, slide-out UI that stays accessible across different app views without interrupting the user's workflow.
*   **Optimized AI Brain:** Powered by **DeepSeek LLM** for high-intelligence reasoning, contextual awareness, and lightning-fast response times at a fraction of traditional API costs.
*   **Real-Time Audio Feedback:** Interactive UI elements/visualizers to indicate listening, thinking, and speaking states.
